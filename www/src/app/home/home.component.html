Penguin Game

TECHNOLOGIES USED

Front-End

Ngx-Rocket 4

This application was created using ngx-rocket, to start the project with correct structure.

Angular 8

Angular is the framework used to build the whole architecture and all the components of the application.

Ionic 4

Ionic is an Angular framework to easily build responsive and thematic components. It was used to define the graphical charter.

Back-End

C++ 17

The game engine was developped in C++, to provide efficient performance.

Emscripten

The C++ code needs to be compile, with Emscripten, in javascript to allow its interpratation into a web browser .

WebAssembly

The game engine is now usable on the web, with WebAssembly.

CONCLUSION

Angular AND WebAssembly

This application is a mix between both technologies, to use the power of WebAssembly directly on a browser and in an Angualar application.
